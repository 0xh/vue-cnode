# vue-cnode

### 技术栈
```
基于vue2 + vue-router + vuex + ES6 + less + flex.css重写vue版cnode社区，使用webpack打包
```

### 功能
- [x] 首页列表，上拉加载
- [x] 主题详情，回复，点赞
- [x] 消息列表
- [x] 消息提醒
- [x] 个人主页
- [x] 用户信息
- [x] 登录
- [x] 退出
- [x] 关于
- [x] 页面后退，数据还原
- [x] 页面后台，滚动位置还原
- [x] ajax请求拦截器
- [x] 页面跳转，不再执行此页面的ajax请求回调方法
- [x] 启动图
